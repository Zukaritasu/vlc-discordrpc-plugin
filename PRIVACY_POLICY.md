# Privacy Policy

This document describes how the **vlc-discordrpc-plugin** project collects, uses, and protects your information.

## 1. Data Controller

- **Project name:** vlc-discordrpc-plugin
- **Controller:** Zukaritasu
- **Contact:** [GitHub profile link](https://github.com/Zukaritasu)

## 2. Information We Collect

**vlc-discordrpc-plugin** does not collect, store, or process any personally identifiable information from end users. The plugin simply connects VLC Media Player with Discord via Discord Rich Presence to display information about the content you are playing locally.

## 3. Use of Information

The only information handled by the plugin is your playback status, and this data is only displayed on your Discord profile through the Rich Presence feature. This data is not sent, collected, or stored on any external server or by the developer.

## 4. Information Disclosure

The plugin does not share information with third parties. All information shown in Discord is managed in accordance with [Discord’s privacy policy](https://discord.com/privacy).

## 5. Security

Since no personal data is stored, no additional security measures are applied. The source code is open and can be reviewed on GitHub for greater transparency.

## 6. Changes to This Policy

This privacy policy may be updated occasionally. Any significant changes will be communicated by updating this document in the official GitHub repository.

## 7. Contact

If you have any questions about this Privacy Policy, you can open an issue or contact via [GitHub](https://github.com/Zukaritasu/vlc-discordrpc-plugin).

---

This policy is provided for informational transparency. If the plugin’s behavior changes and it begins to collect personal information, the policy will be updated accordingly.
